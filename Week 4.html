Week 4: Using CSS

Understanding what makes good website design is a key component in developing my website. Using CSS I can utilize specific styles that I have accrued from my research to develop a website that is easily accessible for the older generation.

I used Coolors to pick my color palette for my website, resulting in a modern look, but one that was readable. I also played around with CSS and Javascript, adding a navigation bar that makes navigating the webpage easy, as well as 'CLICK TO RETURN TO TOP' button, which makes navigating longer pages easier.

Key Points

  • Understanding what makes good website design
  • Using CSS for specific styles
  • Accessibility for the older generation
  • Color palette selection with Coolors
